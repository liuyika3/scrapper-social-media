🔑 用 Gmail OAuth 授权
按 Google Cloud Console 配置好的 OAuth Client，本机直接走 Gmail API 发邮件，不依赖外部进程
前提（已配置过 mail-tool-gmail 可跳过）：
  1. Google Cloud → 启用 Gmail API
  2. OAuth consent screen → 把发件邮箱加进 Test users
  3. Credentials → OAuth client ID（Web app）→ Authorized redirect URIs 加这两条：
    http://127.0.0.1:5180/oauth/callback http://localhost:5180/oauth/callback

凭据写到本机 .google-accounts.json（已加 .gitignore），不会进 commit。
已经在 mail-tool-gmail 用过同一个 Client？直接复用即可，只需在 Cloud Console 给它加 5180 端口的 redirect URI。
